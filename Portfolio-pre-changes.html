    • Task 5

    Accessing data: Scraper and API

    01

    World Bank Data API

    Intentional homicides.

    Base URL: https://api.worldbank.org/v2/country/

    Elements: The API URL appends a list of specific country/region codes separated by semicolons (wld;mex;col;gbr;chl;bra;lcn) followed by the specific indicator code (VC.IHR.PSRC.P5, which represents Intentional Homicides per 100,000 people). It uses query parameters to filter the date range (2010:2023), set the response format to JSON, and increase the per_page limit to 200 to ensure all data points are captured.

    Full URL: https://api.worldbank.org/v2/country/wld;mex;col;gbr;chl;bra;lcn/indicator/VC.IHR.PSRC.P5?date=2010:2023&format=json&per_page=200
